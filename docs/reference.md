# Reference

Primary reference:

- AppLocker App Store listing: https://apps.apple.com/es/app/applocker-passcode-lock-apps/id1132845904

Reference behavior used for this project:

- Password-gated app access.
- Per-app protection.
- Overlay-style lock screen when a protected app is opened.
- Preference surface for password and launch behavior.

Intentionally excluded:

- Touch ID.
- Bluetooth ID.
- Network ID.
- Subscription or upgrade flows.
- Private gallery.
- Private notes.
