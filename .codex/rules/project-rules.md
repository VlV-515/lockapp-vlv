# Project Rules

- Keep the project native Swift.
- Keep all default app UI in English.
- Spanish (Mexico) is optional and must stay user-selectable in Settings.
- Do not add excluded AppLocker Pro-style identity features: Touch ID, Bluetooth ID, or Network ID.
- Do not add monetization or upgrade UI.
- Keep public docs current when commands, packaging, or behavior change.
- Keep the default password documented as `vlv`.
- Treat overlay locking honestly: it is a visual/access gate, not process sandboxing.
